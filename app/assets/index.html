Brunch with nothing but jQuery and a touch of Wisp

As you probably know, Wisp is a cute Clojure-like dialect of Lisp that compiles to Javascript and as such, has made it's way into the web via Brunch
Current example will use function factorial, defined in lib/utils.wisp like so: (def factorial (fn [n] (if (< n 2) n (* n (factorial (- n 1))))))
